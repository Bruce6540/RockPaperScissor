# Rock Paper Scissors Game

## Overview

This project is a simple interactive **Rock Paper Scissors** web game built using HTML, CSS, and JavaScript. The player competes against the computer. The computer randomly selects rock, paper, or scissors, and the game determines the winner based on standard rules.

The website also includes an information section explaining the history, probability, and strategy behind the Rock Paper Scissors game.

A live version of the project is hosted online using Vercel.

---

## Features

* Interactive Rock Paper Scissors gameplay
* Random computer move generation
* Real time result display
* Player and computer score tracking
* Strategy and history section explaining the game
* Responsive layout suitable for basic web viewing

## How the Game Works

1. The player selects rock, paper, or scissors.
2. The computer randomly selects one option.
3. The game compares both selections.
4. The result is displayed as win, lose, or tie.
5. Scores update automatically after each round.

Game rules:

* Rock beats scissors
* Scissors beats paper
* Paper beats rock
